## Оборудование

## Результаты экспериментов
`main.cpp` это программа написанная на C++ с использованием стандарта MPI. Она перемножает две квадратные матрицы, распределяя нагрузку между узлами кластера.
С помошью Putty и WinSCP произведён ввход на суперкомпьютер. Файлы были перенесены и запущены. 
Запуск программы осуществлялся не напрямую, а через систему пакетной обработки заданий SLURM. Был написан batch-скрипт (`start.pbs`)

## Выводы
Суперкомпьютер раскрывает свой потенциал исключительно на огромных объемах данных. Для матрицы 3000x3000 время сократилось с расчётных ~368 секунд (на 1 процессе) до 14.8 секунд (на 20 процессах) и 8.2 секунды (на 40 процессах).
