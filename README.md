Работа сделана Гусаковой В.А. группы 6312-100503D
## Задание
Написать программу на языке C/C++ для перемножения двух квадратных матриц. 
- Входные данные: Файлы, содержащие значения матриц.
- Выходные данные: Файл с результирующей матрицей, время выполнения, объём задачи.
- Верификация: Обязательная проверка результатов с помощью сторонних библиотек (использован Python NumPy).

## Ход работы
Код написан на C++. Матрицы реализованы в виде одномерных векторов `std::vector<int>`. (Файлы а.txt, b.txt, c.txt)

Вычисление реализовано классическими тремя вложенными циклами. Время замеряется с помощью `std::chrono` и учитывает только этап вычислений (файловый ввод-вывод в замер не входит). Верификация и запуск автоматизированы с помощью Python.
